生成

Alt + Enter | 补全提示

Ctrl + w | 选中单词

Ctrl + c/v/d | 复制/粘贴/删除

Ctrl + Shift + Enter | 下一行

导航

Ctrl + Ctrl | 运行命令

shift + shift | 搜索 any

Ctrl + Shift + A | 搜索行为

alt + 1 | 项目窗口

ctrl + E | recent files

ctrl + shift + T | nav to test

alt + shift + </> | 字体放大缩小

重构

  • Ctrl + Alt + M Methods
  • Ctrl + Alt + C Constants
  • Ctrl + Alt + F Fields
  • Ctrl + Alt + V Variables
  • Ctrl + Alt + P Parameters
