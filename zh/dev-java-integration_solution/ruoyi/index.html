RuoYi 后台管理系统

Steven2025年1月1日小于 1 分钟

RuoYi 是一个后台管理系统，基于经典技术组合（Spring Boot、Apache Shiro、MyBatis、Thymeleaf）主要目的让开发者注重专注业务，降低技术难度，从而节省人力成本，缩短项目周期，提高软件安全质量。

官网：

  • https://ruoyi.vip/
  • https://github.com/yangzongzhuan
  • https://gitee.com/y_project/RuoYi

笔记（link_项目地图）：

  • RuoYi-fast —— RuoYi 单应用版本
  • RuoYi —— RuoYi 多模块版本 （RuoYi 从 3.0 开始，进行模块拆分，将原先的单应用转变为多模块）
  • RuoYi-Vue —— RuoYi 前后端分离版本
  • RuoYi-Cloud —— RuoYi 微服务版本
  • RuoYi-App —— RuoYi 移动端版本
