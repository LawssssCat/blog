SpringBoot 介绍

Steven2024年4月5日小于 1 分钟springboot

SpringBoot 目的在于简化 Java 服务的配置、部署。

SpringBoot 核心功能：

  • 简化配置
  • 内嵌 web 服务器
  • 独立运行 java -jar xxx.jar
  • 准生产的应用监控
