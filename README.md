# 博客

```bash
pnpm run docs:dev
```

博客页面路径 <https://theme-hope.vuejs.press/zh/guide/blog/path.html>

`plugins.blog`

| 配置项         | 描述         | 默认路径           |
| -------------- | ------------ | ------------------ |
| `article`      | 文章列表     | `/article/`        |
| `category`     | 分类地图页   | `/category/`       |
| `categoryItem` | 特定分类列表 | `/category/:name/` |
| `tag`          | 标签地图页   | `/tag/`            |
| `tagItem`      | 特定标签列表 | `/tag/:name/`      |
| `star`         | 星标文章列表 | `/star/`           |
| `timeline`     | 时间线列表   | `/timeline/`       |
