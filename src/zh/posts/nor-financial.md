## 日本

参考：

+ 冷水TV|日本财团介绍 - <https://www.youtube.com/watch?v=lth_yyRNpPM>

财团御三家：以家族经营方式在各个领域控股的大型垄断集团。通过资本力量影响国家经济政策和社会结构。

::: tip
财团 —— 原“财阀”，二战战败后资产被冻结，子公司被拆分有各自自主权，但为了经济复苏抱团取暖，形成如今“财团”。
:::

+ 三菱（MITSUBISHI） —— | 路径： 公务员（信息） => 倒爷（资金、人脉） => 海运（技术） => 矿业（海运能源）、保险（海运风险）、仓库（海运物流） => n | 产品： 三菱商事、三菱重工、三菱UFJ银行、三菱汽车、三菱电器、麒麟啤酒、罗森、...
+ 三井（MITSUI） —— | 路径： 和服（资金） => 金融（政商） => 风投（扎根各行各业） => n | 产品： 三井物产（基建）、三井不动产（基建）、三井住友金融集团、工业、矿业、丰田（TOYOTA）、东芝（TOSHIBA）、三越百货（MITSUKOSHI）、富士胶卷、札幌啤酒、东京巨蛋、...
+ 住友（SUMITOMO） —— | 路径： 天皇曾孙（信息、人脉） => 炼铜（资金、技术） => 矿业、金融业（政商） => n | 产品： 住友商事、住友电工、日本电器（NEC）、川崎重工、大阪商船、雅马哈发动机
+ 富士（FUJI） —— 产品： 日产（NISSAN）、日立（HITACHI）

> 众所周知，御三家有四个。
