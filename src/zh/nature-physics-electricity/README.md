地线、零线、火线

- 对于设备

  ![设备：地线、零线、火线](https://files.catbox.moe/22pc42.png)

  - 地线 —— 连接设备外壳
  - 零线 —— 连接电路板一侧
  - 火线 —— 连接电路板另一侧

- 对于供电商

  ![供电商：地线、零线、火线](https://files.catbox.moe/jln3w7.png)

  - 地线 —— 社区重新接地后提供的零电位线，即N（Neutral，中位线，俗称“零线”）
  - 零线 —— 社区重新接地后提供的零电位线，即PE（Protective Earthing，保护性接地，俗称“地线”）
  - 火线 —— 社区从供电商获取的220V通电线，即L（Live，通电线，俗称“火线”）

漏电场景：由于污渍、积水等原因，导致火线与设备金属外壳接触

- 有地线

  ![漏电场景：有地线](https://files.catbox.moe/2ethzs.png)

  这种情况下，相当于火线直接接地，会产生220v电位差，产生大电流。
  大电流会导致电闸跳闸。
  由于人体电阻远大于导线电阻，所以电流在并联分流后只有极少电流流经人体，所以人体不会有触电的感觉。

  ```bash
                ┌───> [ 电路板 ] ───> 零线(0V)  ─────────────> ⚡ 正常工作电流
                │
  火线(220V) ───┤
                │                   ┌──> [ 地线 ](电阻 4Ω) ──> 大地(0V) ⭐ 99.9% 漏电走这里
                └───> [ 金属外壳 ] ──┤
                                    └──> [ 人体 ](电阻 2000Ω) ──> 大地(0V) ❌ 几乎无电流
  ```

- 无地线

  ![漏电场景：无地线](https://files.catbox.moe/vkq32r.png)

  没有地线分流，电流全部流向人体

  ```bash
                ┌───> [ 线路电阻小 ] ──> [ 电路板 ] ──> 零线(0V) ───=> ⚡【通路1】正常工作
                │
  火线(220V) ────┤
                │
                └───> [ 污渍电阻 ] ──> [ 金属外壳 ] ──> [ 人体 ] ──> 大地(0V) ──=> 🔥【通路2】人触电
  ```
